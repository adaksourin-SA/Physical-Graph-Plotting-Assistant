# Physical Graph Plotting Assistant

A Python-based GUI tool designed to assist physics students in plotting accurate laboratory graphs using physical graph sheets.

## Features

- Automatic scale calculation
- Large & small square conversion
- Axis marking generation
- Physical graph location assistant
- Value ↔ Graph conversion
- Manual slope calculation (triangle method)
- Visual slope triangle overlay
- Grid customization (major & minor ticks)

## Built With

- Python
- Tkinter
- Matplotlib
- NumPy

## How to Run

1. Clone the repository:
   git clone https://github.com/adaksourin-SA/Physical-Graph-Plotting-Assistant.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python main_GUI.py

## Use Case

Designed for physics laboratory practical work to reduce manual scale calculation errors and improve plotting accuracy.
